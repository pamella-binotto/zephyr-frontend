# 🌤️ Zephyr Frontend

Interface web da plataforma **Zephyr**, desenvolvida em React para consumir a API do projeto Zephyr.

## Sobre o projeto

O Zephyr Frontend será responsável por fornecer uma interface moderna e intuitiva para que os usuários possam:

* Realizar autenticação utilizando JWT;
* Consultar o clima atual;
* Visualizar previsões meteorológicas;
* Gerenciar cidades favoritas;
* Receber recomendações inteligentes geradas por Inteligência Artificial.

## Tecnologias

* React
* Vite
* Tailwind CSS
* React Router DOM
* Axios

## Status do projeto

🚧 Em desenvolvimento.

Este repositório está sendo desenvolvido em conjunto com a API do Zephyr, seguindo boas práticas de organização, componentização e desenvolvimento moderno com React.
